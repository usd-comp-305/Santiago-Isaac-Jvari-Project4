# Dungeon Daedalus

## Description

Dungeon Daedalus is a text-based dungeon adventure game written in Java. The player explores a dungeon, fights enemies, earns gold, buys upgrades from a shop, uses potions, and progresses through harder difficulty levels.

The goal of this project is to practice programming and software design. The project uses inheritance, polymorphism, abstraction, encapsulation, interfaces, design patterns, unit testing, Gradle, and GitHub.

## Features

- Player can enter a name and play through a text-based menu
- Player can enter and leave the dungeon
- Player can explore left, forward, or right
- Different enemies can appear during exploration
- Boss enemies increase the dungeon difficulty when defeated
- Player can fight using different attack strategies
- Player can buy weapon upgrades, defense upgrades, and potions from the shop
- Player can use health and strength potions
- Game tracks final stats such as:
    - Total gold
    - Enemies defeated
    - Bosses defeated
    - Rooms explored
    - Difficulty reached

## How to Run the Project

This project uses Gradle. Make sure you are in the root folder of the project.

To run the game:

```bash
./gradlew run

To run all tests:
./gradlew test

```
